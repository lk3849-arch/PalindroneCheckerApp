# PalindromeCheckerApp

// Project Name: Palindrome Checker App

// Running Procedure:
// 1. Open the project in any Java IDE (Eclipse, IntelliJ, VS Code)
//    OR use command prompt.
// 2. Compile the file:
//      javac PalindroneCheckerApp.java
// 3. Run the program:
//      java PalindroneCheckerApp
// 4. The system will display a welcome message.

// Flow of Project:
// Step 1: Application starts.
// Step 2: Main method executes.
// Step 3: Welcome message is displayed on the console.
// Step 4: Program terminates.

// Objective:
// The objective of this project is to build a simple
// Palindrome Checker Management System using Java.
// Currently, it displays a welcome message.
// Future versions will check whether a word or sentence
// is a palindrome.

// Topics Covered:
// - Java Basics
// - Main Method
// - Class Structure
// - Console Output (System.out.println)
// - Basic Program Execution Flow

// Use Case:
// This application can be used:
// - To check whether a word or number is a palindrome.
// - For learning basic Java programming.
// - As a beginner-level academic project.
// - As a foundation for expanding into a full palindrome checking system.
